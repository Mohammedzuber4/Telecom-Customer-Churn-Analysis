# 📊 Telecom Customer Churn Analysis

## Project Overview
An end-to-end data analytics project to identify factors influencing customer churn in the telecom industry using Python, SQL, and Power BI.

## Tech Stack
- Python (Pandas, Matplotlib, Seaborn)
- SQL (MySQL)
- Power BI
- DAX

## Dataset
- Telco Customer Churn Dataset
- 7,032 customer records
- 21 features

## Project Workflow
CSV Dataset
↓
Python (Cleaning + EDA + Visualization)
↓
SQL Analysis
↓
Power BI Dashboard
↓
Business Insights

## Key Insights
- Month-to-month customers showed the highest churn.
- Customers with higher monthly charges were more likely to leave.
- Customers with lower tenure exhibited higher churn.
- Electronic check users demonstrated higher churn.

## Business Recommendations
- Promote long-term contracts.
- Create retention campaigns for new customers.
- Monitor high-billing customers.
- Improve engagement strategies for high-risk segments.
